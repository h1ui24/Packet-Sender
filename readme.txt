============================================
Packet Sender - UDP/ICMP 패킷 전송 도구
버전: 1.0
제작자: h1ui24
============================================

[프로그램 개요]
- 이 도구는 네트워크 테스트를 위해 UDP 또는 ICMP 패킷을 전송할 수 있습니다.
- Snort나 IPS 탐지 테스트, 네트워크 발생 실험에 활용 가능합니다.

[프로그램 실행]
- Windows에서 실행: packet_sender.exe
- 설치 불필요, 더블클릭으로 바로 실행됩니다.

[입력 항목 설명]
- 목적지 IP: 패킷을 보낼 대상 IP 주소 (예: 192.168.0.10)
- 출발지 포트: 패킷의 출발지 포트 번호 (예: 12345)
- 목적지 포트: 패킷의 목적지 포트 번호 (예: 53)
- 프로토콜: UDP 또는 ICMP를 선택
- Payload (hex):
  예시: \x9f\x4f\x00\x00\x00\x00\x00\x00\x27\x0f\x61\x62\x63
  - \x 형식으로 입력 (각 바이트마다 \x 붙임)
  - Snort 룰과 동일한 형태의 Payload를 입력
- [패킷 전송하기] 버튼: 입력값으로 패킷을 전송

[주의사항]
- 보안 소프트웨어나 방화벽에서 경고가 발생할 수 있습니다.
  (Scapy 기반으로 Raw 패킷을 전송하기 때문)
- 반드시 네트워크 테스트 목적으로만 사용하세요.

[지원]
- 문제가 발생하거나 수정/개선 요청은 제작자에게 문의주세요.
